# walt-quick-start
a demo for those guys who want to experience [walt](https://github.com/ballercat/walt) in a simply way

## Install

`npm install`

## Build
`npm run build`

## Load result
View index.html in your browser, if everything goes well, the result below should appear at the console.

```
1
2
1
```
